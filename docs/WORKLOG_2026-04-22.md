# Worklog — 2026-04-22

Este documento resume as alterações feitas no projeto **MechUp-app** hoje (backend + app).

## Login / Auth

- Corrigimos persistência de sessão na **web**: `expo-secure-store` não funciona no web, então o token passou a ser salvo em `localStorage` no web e continua em SecureStore no mobile.
- Melhoramos o login no backend para lidar com **e-mail repetido em mais de uma oficina**:
  - o backend testa a senha em todos os candidatos daquele e-mail e prioriza uma oficina **ativa**.
- Melhoramos mensagens de erro no login (rede/API/credenciais).

Arquivos principais:
- `src/auth/AuthProvider.tsx`
- `server/routes/auth.ts`
- `app/login.tsx`
- `lib/api.ts`
